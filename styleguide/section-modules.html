4Modules #permalink

Code snippets that come with the brink tech SMACSS package

4.1bubbles #permalink

creates independent content sections that hover above the page in a sort of bubble

Example 
<div class='bubbles' style='width: 30em;'><div class='bubble rounded-corners drop-shadow'></div><div class='bubble rounded-corners drop-shadow'></div><div class='bubble rounded-corners drop-shadow'></div><div class='bubble rounded-corners drop-shadow'></div></div>

4.2drop shadow #permalink

Creates a drop-shadow effect on any elements with class 'drop-shadow'

Example 
<div class='drop-shadow' style='width: 10em; height: 10em;'></div>

4.3form: two-column #permalink

creates a two-column form empty spaces can be filled with .label-placeholder

Example 
<form class='two-column'>
	<label>name</label><input type='text'></input>
	<label>email</label><input type='email'></input>
	<div class='label-placeholder'></div><input type='submit'></input>
</form>
